📖 SETUP GUIDE
// HOW TO DEPLOY ON GITHUB PAGES
▸ INITIAL SETUP
1. Create a GitHub repo named yourusername.github.io
2. Upload all 3 files: index.html, admin.html, content.json
3. In repo Settings → Pages → Source: Deploy from branch: main / root
4. Your site goes live at https://yourusername.github.io
▸ UPDATING CONTENT
1. Open admin.html locally (or at yourusername.github.io/admin.html)
2. Edit whatever you want in the form panels
3. Click 💾 EXPORT JSON to download the updated content.json
4. Commit & push the new content.json to GitHub
5. GitHub Pages auto-deploys in ~30 seconds ✓
▸ COLOR CODES
"" (empty) = Green accent
"a" = Amber / orange accent
"c" = Cyan / blue accent
"d" = Dim / neutral (for chips)
▸ TIPS
• Use LOAD JSON to bring in an existing content.json to keep editing it
• The RAW JSON panel lets you paste/edit JSON directly
• Admin panel works 100% offline — no server needed
• Contact form needs a service like Formspree.io (free) for real emails
